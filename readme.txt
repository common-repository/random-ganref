Random Ganref is sidebar widget to display images from Ganref.jp service (similar with Flickr).

== Description ==

A sidebar widget to display images from ganref server with optional thumbnails. Ganref.jp is one of image storage service like flickr for especially DSLR user in Japan.

This widget is approved by ganref.jp.

== Installation ==

Place the downloaded folder in your /wp-content/plugins/ directory
and activate through the administration panel, and then go to the widget panel and
drag it to where you would like to have it!
